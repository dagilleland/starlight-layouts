import { DashboardGrid, Widget } from '@starlight-layouts/layout-dashboard';

This page sets `pageLayout: dashboard`. The
[`TwoColumnContent` override](/starlight-layouts/tutorial/full-width-layout/) drops the
right-hand table of contents, same as [full-width](/starlight-layouts/demos/full-width/), and
the [`ContentPanel` override](/starlight-layouts/tutorial/dashboard-layout/) removes
Starlight's usual content-width cap so the grid below can spread across the
full main pane. The left navigation sidebar stays, so this page is still
reachable and browsable like any other.

</DashboardGrid>

and when a number needs context, see
[Filling a dashboard](/starlight-layouts/guides/dashboard-widgets/) under **Guides**.
